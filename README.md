# Lacquer & Lily (demo)

A demo website for a fictional Louisville nail studio. Fashion-editorial design: blush base, rose-gold accent, didone display type, a spa-card service menu, and a lookbook gallery grid.

Live at [davin-willis.github.io/nailsalon-demo](https://davin-willis.github.io/nailsalon-demo/). Part of my client web work at [davinwillis.dev](https://davinwillis.dev).

## AI-assisted development

Built with AI-assisted development. All architecture, design, and implementation decisions are my own.

## Notes

The business is fictional. All names, phone numbers, addresses, and details are placeholders, and photo slots are clearly marked placeholders.

## Stack

- Next.js (App Router, TypeScript, static export)
- Tailwind CSS v4
- Framer Motion (soft fade reveals, respects prefers-reduced-motion)
- Fonts: Bodoni Moda, Jost via next/font
- Deployed to GitHub Pages via GitHub Actions

## Run locally

```bash
npm install
npm run dev
```
